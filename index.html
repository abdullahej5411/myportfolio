Selected work

Five projects spanning full-stack web platforms and applied machine learning - from live production apps to research-grade AI systems.

Live · AI Web App

Tensor Film | Smart Movie Explorer | MERN Stack | Next.js

An AI movie discovery platform that classifies user intent - mood, specific title, or trivia - using Gemini AI, then cross-references TMDB for movie data and caches results in Redis for performance. Backend built with Express and Node.js, handling JWT-based authentication and MongoDB-backed user profiles for watchlists and favorites. The Next.js/TypeScript frontend renders recommendations with posters, trailers, and an interactive cinema mode. Fully containerized with Docker for local development, with the backend deployed on Render and the frontend on Vercel.

Next.jsExpress.jsMongoDBRedisGemini AIRender + Vercel
Final Year Project

BioMechAI | Real-Time Fitness Trainer | Flutter App

A real-time AI fitness trainer that tracks 33 body landmarks at ~30 FPS via MediaPipe, using a custom Random Forest classifier to recognize 7+ exercises from 1,050+ training videos at ~85% confidence. The Flutter app handles automatic exercise detection, rep counting, and weight logging, syncing sessions to Firebase. A Flask backend on Render extracts 21 joint-movement features and serves classification via REST API, while a React trainer dashboard lets certified trainers review AI-annotated sessions and send direct feedback - built end-to-end across a 3-member team.

FlutterMediaPipeRandom ForestFlaskReactFirebase
Deep Learning

Neural Storyteller | Image Captioning Pipeline | ResNet50 + LSTM

An image-to-text deep learning pipeline: a pre-trained ResNet50 extracts a 2048-dimensional feature vector from each image, decoded by a multi-layer LSTM into a word-by-word caption. Trained on Flickr30k (31,000 images, 5 captions each) using teacher forcing, with Beam Search (k=5) at inference for more descriptive, less repetitive output. Deployed via Gradio for real-time caption generation from user-uploaded images.

ResNet50LSTMFlickr30kBeam SearchGradio
